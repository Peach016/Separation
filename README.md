# Separation
Separation - convenient program for separate text.


## Documentation
In Separation - one function ```separation()```.

This function takes three arguments:

Text data, part width and separator

Example:
```Python
hexdata = "ff000000ff000000ff"
print(separation(text=hexdata, width=6, separator=" "))
# Output: "ff0000 00ff00 0000ff"
```
